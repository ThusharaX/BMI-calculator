# BMI-calculator

BMI-calculator is a Python progrme for calculate BMI

## Requirements

Python 3.8.2
[Download](https://www.python.org/downloads/release/python-382/)

## Installation

```bash
git clone https://github.com/ThusharaX/BMI-calculator.git
cd BMI-calculator
```
## Usage

```bash
python main.py
```
